Scan for the nearest Starfleet vessel and take command of the bridge.

📡 Compatibility

💻 Windows ✓ Chrome or Edge — works out of the box. Needs Bluetooth adapter.
🍎 macOS ✓ Chrome or Edge. Brave not supported (Web Bluetooth blocked).
🐧 Linux ✓ Chrome with chrome://flags/#enable-web-bluetooth enabled.
🤖 Android ✓ Chrome — works natively, no extra setup needed.
📱 iOS ⚠ Safari & Chrome don't support Web Bluetooth. Install Bluefy (Web BLE browser, ~$2). Then either:
① AirDrop this HTML file to your iPhone → open in Bluefy from Files app, or
② On your computer, run python3 -m http.server 8000 in the folder with this file → on iPhone in Bluefy, go to http://<your-computer-ip>:8000/enterprise-controller.html (both devices must be on the same Wi-Fi).
Your iPhone's Bluetooth connects directly to the ship — the computer just serves the file.
